# Data Visualization

## Description

This is a project repo for data visualization in D3.js and Dimple.js.

## What is in this repo

D3.js 
* Bubble chart
* Bubble map 
* Animated bubble map (author-driven and user-driven)
* Parelle Coordinates with tooltip

Dimple.js
* Line chart
